## Setup

The following _should_ work:
1. Make sure you have uv installed
2. Open the project in your favorite IDE
3. Ensure that there exists a virtual environment that is managed by `uv`
4. Run `uv sync` to update that environment from `pyproject.toml 

Now you should be able to run jupyter notebook etc from within your IDE
Adding python modules/functions in `src/parlianets` should automatically make them available for you to import in notebooks etc.

When in doubt, ask your group leaders for help!
